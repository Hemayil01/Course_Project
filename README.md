# Course - Təhsil Platforması

Course System - Django layihəsidir və istifadəçilərə kursları izləməyə, qeydiyyatdan keçməyə və öyrənmə prosesini idarə etməyə imkan verir. Sistemdə 'student' yalnız kurslara baxa və qeydiyyatdan keçə bilər, 'instructor' kurs və dərsləri idarə edir, tələbələrin irəliləyişini izləyir, 'admin' isə tam idarəetməyə malikdir. Layihə avtomatik olaraq kursun müddətini hesablayır və istifadəçinin kursda irəliləyişini (progress) izləyir.

The Course System is a Django-based project that allows users to explore courses, enroll in them, and track their learning progress. In the system, a 'student' can only view courses, enroll, and track their progress, an 'instructor' manages courses and lessons, monitors student progress, while an 'admin' has full control over the entire system. The project automatically sets course duration and tracks user progress based on completed lessons.
