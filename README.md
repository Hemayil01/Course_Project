# Course - Təhsil Platforması

Course System Django REST Framework istifadə edilərək hazırlanmış kurs idarəetmə sistemidir və istifadəçilərə kursları izləmək, kurslara yazılmaq və öyrənmə prosesini idarə etmək imkanı verir.
Sistemdə istifadəçilər rollara bölünür:
- 'student' kurslara baxa, kurslara yazıla və öz öyrənmə irəliləyişini izləyə bilər
- 'instructor' kurs və dərsləri yaradır və idarə edir, tələbələrin irəliləyişini izləyir
- 'admin' isə sistem üzərində tam idarəetməyə malikdir
Layihə avtomatik olaraq kurs müddətini və istifadəçinin kurs üzrə irəliləyişini (progress) hesablayır.

# Course - Learning Platform

The Course System is a Django REST Framework-based project that allows users to browse courses, enroll in them, and track their learning progress.
The system includes role-based access:
- 'student' can view courses, enroll, and track their learning progress
- 'instructor' manages courses and lessons and monitors student progress
- 'admin' has full control over the entire system
The project automatically calculates course duration and tracks user progress based on completed lessons.
